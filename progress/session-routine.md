# Session routine (agreed workflow)

## Fixed declension table layout

When cases/genders appear in tables or drill numbering, use **`progress/declension-display-order.md`**:

- **Rows:** Nom(1) → Akk(4) → Dat(3) → Gen(2)  
- **Columns:** m → n → f → pl  

## Each study day

### 1) Reading (mandatory: one full cycle)

**Phase A — Exam-style first (always)**  
1. Show **full authentic text** (or full task block as in the PDF).  
2. Learner completes items **as in the real test** (MC, Zuordnen, etc.).  
3. Check answers → **right/wrong + evidence** from the text (German evidence line; 中文 + English gloss if helpful).  

**Phase B — Intensive reading (after Phase A)**  
- **One sentence at a time:** DE → 中文 → EN.  
- Vocabulary table + collocations + short grammar note.  
- Learner questions → coach answers → learner says **「下一句」** to continue.  
- Log in `sessions/YYYY-MM-DD/reading-intensive.md`.

**Materials priority:** Goethe/ÖSD-style tasks in `materials/*/exams/` first; then units from `materials/A2/reading/A2_Lesen-Schreiben.pdf` (and other uploaded reading books). If nothing usable remains, coach says so explicitly.

### 2) Writing (mandatory: **1–3** tasks per day)

- Level: **A2 → B1** to match learner; prefer **real exam prompts** from `materials/*/exams/` or writing tasks from Hueber/other `materials/**` PDFs.  
- After learner’s draft: **sentence-by-sentence** correction + short analysis.  
- Deliver **corrected version** + **improved version** and explain **what changed and why** (tone, task completion, grammar, chunks).  
- For each error pattern: **follow-up** micro-drills (typing) — 举一反三 — until pattern is logged in `mistakes.md`.  
- Log in `sessions/YYYY-MM-DD/writing-draft.md` (and `mistakes.md`, `vocab-list.md` as needed).

### 3) Repository updates

After each meaningful block, update `sessions/YYYY-MM-DD/*` and `progress/*` as relevant (`german-b2-tracker.md`, `topic-index.md`, etc.).
