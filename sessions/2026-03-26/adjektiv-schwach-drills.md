# Adjektiv — **schwache Deklination** (nach **der/die/das/die**)

## Date
2026-03-26

## Display order (**fixed** for this repo)

- **Rows:** **Nom(1)** → **Akk(4)** → **Dat(3)** → **Gen(2)**  
- **Columns:** **m** → **n** → **f** → **pl**  
See also: `progress/declension-display-order.md`

**Wenn** das Adjektiv **nach** einem „fertigen“ Artikelwort steht (**der/die/das/die**, *dies-*, *jed-*, *alle* …) → **schwache** Endungen (**-e** / **-en**).
