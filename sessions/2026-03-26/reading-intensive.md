## Sentence 1

**DE:** Bei Stefan Berger gibt es Gerichte, von denen man vorher noch nie gehört hat.

**中文：**在斯特凡·伯杰（这儿/这家餐厅），有（一些）人们以前从未听说过的菜肴。

**EN:** At Stefan Berger’s [restaurant], there are dishes (that) people have never heard of before.

### Vocabulary (sentence lemmas)

| Wort | 中文 | English |
|------|------|---------|
| bei | 在……处；在……那里 | at; with (location/person) |
| geben (es gibt) | 有；存在 | there is/are |
| das Gericht, -e | 菜肴；一道菜 | dish (food) |
| von | 从；关于 | from; of |
| der/die/das, die (Pl.) | （关系从句里指代 Gerichte） | which |
| vorher | 事先；以前 | before; previously |
| noch nie | 从未 | never (yet) |
| hören | 听；听说 | to hear |
| man | 人们；有人 | one; people |

### Phrases / collocations

- **es gibt + Akk.** — there is/are …  
- **von etwas gehört haben** — to have heard **of** something  
- **noch nie** — never (emphatic)  
- **vorher** — beforehand / before that  

### Grammar note (short)

- **„von denen“** = relative pronoun (**die** + **von**) referring back to **Gerichte** (plural): *Gerichte, von denen …* = “dishes **of which** … / dishes **that** …”.  
- Verb at end of relative clause: **… gehört hat** (Perfekt of *hören*; **haben** auxiliary).
