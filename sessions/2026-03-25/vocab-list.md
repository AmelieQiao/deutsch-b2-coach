# Vocab List (Baseline Round 2)

## Date
2026-03-25

### From reading (Hausordnung)

- **verbindlich**
  - **中文**：有约束力的 / 必须遵守的  
  - **English**: binding / mandatory  
  - **example (DE)**: Die Unterrichtszeiten sind verbindlich.  
  - **grammar & memory note**: oft in Regeln / Verträgen  

- **untersagt**
  - **中文**：禁止的  
  - **English**: forbidden / prohibited  
  - **example (DE)**: Es ist untersagt, an die Wände zu kleben.  

- **In Ausnahmefällen**
  - **中文**：在例外情况下  
  - **English**: in exceptional cases  
  - **example (DE)**: In Ausnahmefällen kann die Schulleitung den Konsum von Alkohol erlauben.  
  - **grammar & memory note**: Gegenpol zu „ohne Ausnahme / immer verboten“ — Lesetrick merken  

- **zur Verfügung stehen**
  - **中文**：可供使用  
  - **English**: to be available  
  - **example (DE)**: Schließfächer stehen kostenlos zur Verfügung.  

### Formal email (Frau Müller)

- **Ihre Einladung**
  - **中文**：您的邀请（尊称）  
  - **English**: your invitation (formal)  
  - **grammar & memory note**: **I-** groß bei Sie-Form; **Einladung** Substantiv groß  

- **Ich habe Ihre Einladung erhalten.**
  - **中文**：我已收到您的邀请。  
  - **English**: I have received your invitation.  
  - **grammar & memory note**: Perfekt: *haben* + **erhalten** (nicht *habe bekommt*)  

- **Es tut mir leid, dass …**
  - **中文**：很遗憾……  
  - **English**: I’m sorry that …  
  - **grammar & memory note**: **dass**-Satz → konjugiertes Verb oft **am Ende**: *dass ich heute nicht kommen **kann***  

- **Vielleicht könnten wir uns treffen.**
  - **中文**：也许我们可以见面。  
  - **English**: Perhaps we could meet.  
  - **grammar & memory note**: **treffen** (见面) ≠ **trennen** (分开)  

- **Mit freundlichen Grüßen**
  - **中文**：此致敬礼（正式）  
  - **English**: Kind regards (formal)  
  - **grammar & memory note**: **Grüße** mit **ü**
